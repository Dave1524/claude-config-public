# Claude Configuration Files

This repository contains Claude AI configuration files and skills for development workflows.

## Contents

- `.claude/` - Claude IDE configuration and skills
  - `claude_skills/` - Various development skills and workflows
  - `settings.local.json` - Local settings
- `CLAUDE.md` - Claude configuration documentation

This repository is made public to allow AI assistants (like Grok) to analyze and understand the Claude configuration structure.
